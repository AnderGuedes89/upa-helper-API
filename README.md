# UPA Helper - API

### Descrição

API de uma plataforma de gestão em saúde com foco em pequenos hospitais com pronto atendimento para auxiliar nas operações da recepção, consulta medica, ambulatório, farmácia, laboratório, radiologia e estoque.

### Tecnologias

- [NestJS](https://nestjs.com/)
- [Node.js](https://nodejs.org/en/)
- [PostgreSQL](https://www.postgresql.org/)
- [Sequelize](https://sequelize.org/)
- [JWT](https://jwt.io/)
- [Passport](https://github.com/jaredhanson/passport)
- [Bcrypt](https://www.npmjs.com/package/bcrypt)
- [Swagger](https://swagger.io/)

### Instalação

```bash
$ npm install
```

### Rodando localmente

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
